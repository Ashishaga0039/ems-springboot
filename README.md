Employee Risk Detection System

The Employee Risk Detection System is a full-stack web application designed to help organizations efficiently manage employee information and monitor potential employee risk through an interactive dashboard. The application enables administrators to perform complete employee management operations, analyze employee data, and visualize key insights for better decision-making.

Key Features
Employee Management (Create, Read, Update, Delete)
Employee Risk Analysis Dashboard
Interactive Analytics and Data Visualization
RESTful API Architecture
Responsive User Interface
PostgreSQL Database Integration
Layered Backend Architecture using Spring Boot
